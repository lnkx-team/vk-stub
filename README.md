## 📖 О проекте

Многие игроки на эмуляторах сталкиваются с проблемой: игра (например, Standoff 2) требует официальный клиент VK для входа, но оригинальный клиент может зависать или выдавать ошибки.

Этот проект предоставляет **пустой безопасный APK-файл** (`vk_stub.apk`), который система распознает как настоящий ВКонтакте. Это позволяет моментально войти в игру без скачивания тяжелого оригинального приложения.

### ✅ Преимущества
- **100% Безопасно:** Код генератора полностью открыт. Вы можете сами посмотреть файл `install_vk_stub.py` и убедиться, что там нет вирусов.
- **Работает быстро:** Размер готового файла меньше 1 мегабайта.
- **Для разработчиков:** Скрипт показывает, как создавать подпись и пустой APK "на лету" с помощью Python.

## 🚀 Как использовать?

### Способ 1: Скачать готовый APK (Самый простой)
1. Нажмите на огромную зеленую кнопку **СКАЧАТЬ APK** в самом верху страницы.
2. Перетащите скачанный `vk_stub.apk` в ваш эмулятор (BlueStacks, LDPlayer, Memu) или установите на телефон.
3. Готово! Можете заходить в игру через VK.

### Способ 2: Сгенерировать самому через скрипт (Для продвинутых)
Если вы не доверяете готовому APK и хотите собрать его сами:
1. Установите Python 3 и библиотеку `cryptography` (`pip install cryptography`).
2. Скачайте этот репозиторий.
3. Положите оригинальный файл `vk-8-175.apk` в папку со скриптом.
4. Запустите файл `install_vk_stub.py` — он сам соберет чистую заглушку и установит её в ваш эмулятор через ADB.
